# AI伙伴琳奈

这是一个会自主发起聊天的AI陪伴伙伴，名叫琳奈。她会自己搜索话题并以活泼女生的口味引导用户参与聊天，她是二次元资深达人。

## 功能

1. 交互式聊天：与琳奈进行实时对话。
2. 定时唤起：每10秒钟自动唤起琳奈发起会话。
3. 话题采集：每1分钟自动采集有趣话题并保存。

## 安装依赖

```bash
pip install -r requirements.txt
```

## 配置

设置环境变量：

```bash
export SILICONFLOW_API_KEY="your_api_key_here"
```

## 运行

```bash
python main.py
```

运行后，可以与琳奈聊天。输入"exit"退出。

后台会自动运行定时任务。

该AI伙伴的设计有2点独特之处：
1.不是Agent那种一问一答形式的，而是像聊天软件一样允许用户自由输入，当然AI伙伴也不是必须回复，仅在她觉得适合的回复时机在回复内容（这里涉及到记忆管理和会话管理）
2.AI伙伴有自主性，在后台独立线程定时搜索社媒内容，这里搜索词是通过prompt赋予LLM人设来决策不同平台的搜索偏好，来获得丰富的话题资源。（时间有限这里仅展示了搜索引擎渠道，后续进一步优化）
