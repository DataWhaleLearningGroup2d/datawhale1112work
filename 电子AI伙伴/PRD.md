# 本项目是一个AI陪伴伙伴
# description: AI伙伴名叫*琳奈*，是一个能够主动发起会话的Agent。她具备定时唤起和主动会话2种模式。在空闲时段会自主在后台搜索社交媒体的内容，用”25岁热爱生活、热情活泼的女生的用户偏好“，自主摘录社交媒体上有趣的内容并生成”话题“，在用户主动发起会话或者定时唤起*琳奈*时，*琳奈*能够像好友那样和用户聊天，并分享今天的见闻。
# 功能list:
1. 交互式聊天功能：通过预先配置的prompt为*琳奈*赋予人设，并启动主Agent，通过浏览器打开一个网页作为对话框，能和*琳奈*主动聊天。
2. 定时唤起功能:后台启动1个定时调度程序，设定为每15分钟(开出配置项可以让开发者自行配置)主动唤起*琳奈*发起会话，通过浏览器的对话界面和用户聊天。
3. 话题采集功能：后台启动另1个定时调度程序，设定为每10分钟搜索网页或社交媒体为*琳奈*搜集聊天话题，*琳奈*的搜索行为和点击行为偏好需要通过预定义的prompt作为指引，搜索到5条新鲜事后结束话题当前采集任务，采集到的话题按时间作为文件名存储到本地磁盘，当和琳奈聊天时，琳奈会检索当天的话题载入到Agent的上下文。
